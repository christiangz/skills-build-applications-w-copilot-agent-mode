The React frontend uses `import.meta.env.VITE_CODESPACE_NAME` to build the backend API host for GitHub Codespaces.
Define `VITE_CODESPACE_NAME` in `octofit-tracker/frontend/.env.local` or use `octofit-tracker/frontend/.env.local.example`.
When that variable is unset, the frontend falls back to `http://localhost:8000`.
